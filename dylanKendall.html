Project Role & Contributions

I served as the primary backend and full-stack developer for eBookSmith. I handled nearly all JavaScript logic, built the eBook generation system in PHP, integrated the ChatGPT API for story/title generation, and implemented SQLite-based transaction logging. On the frontend, I developed the Forge and Library pages, built the dynamic navbar/footer system, and designed the site’s glowing theme and branding.

Challenges & Solutions

One of the biggest challenges was getting the EPUB generation to work with valid XHTML content. I overcame this by researching formatting requirements and using AI tools to help generate clean markup. Styling tricky components like accordions also took time and experimentation to get just right—but the result was worth it.

Reflections

I also learned that I genuinely enjoy making things look good—and that I'm good at it.
